Settings

Chrome Downloads Bar

Off means download bar is disabled.

Download Completion Notification

On means notifications are enabled when download completes

On Notification Click, Do

select what u want to do when u click on notification

Options

Clear Failed

This Option will clear all downloads that eighter failed due to network error or cancelled by the user

Clear Deleted

This Option will clear all downloads whom files have been deleted form disk

Clear Completed

This Option will clear all downloads which are successfully completed (be it on the disk or not). This doesn't clear cancelled downloads

What More?
